# hacker-hub
> A starter hub for RBOIS hacking

## Communities
- OWASP: A community of people focused on improving the state software security
- DEF CON Forums: A community for people who are interested in hacking and security research to meet up.
- FIRST: A community for security and incident response professionals to share information.
- Red Team Village: A community group that focuses only on red teaming and offensive security.
- Infosec.Exchange: A community centered around information security.

## Researchers
- SpecterOps: Publishes technical red teaming research.
- Mandiant Group: Provides information pertaining to adversarial behavior, and intrusion trends.
- CISA: Publishes findings and common weaknesses found during assessments.
- DFIR Report: A collection of real-world incident response reports.

## Writeups/Case Studies
- The DFIR Report Case Studies: These show how real cyber attacks happened in great detail.
- PortSwigger Web Security Research: Writes about new ways to attack the web and how to use discovered weaknesses as weapons.
- SpecterOps Blog: Publishes research articles about advanced attacks that happen in business.
- SANS Reading Room: Publishes in-depth technical papers that cover a wide range of cybersecurity issues.
## Events/Conferences
- DEF CON: A hacker conference where people talk about the newest tools, exploits, and research.
- Black Hat: A conference that gives people training and technical briefings on cybersecurity. 
- Red Team Village Content: A collection of talks and recorded training materials from events held by Red Team Village. 

## Rabbit Holes
- Null Byte: A site with hands-on hacking tutorials, walkthroughs, and guides that focus on app exploitation, Wi-Fi hacking, OSINT, and social engineering (Facebook).

#### Advanced Red Team Tools
- PowerUpSQL: A tool for finding, checking, and taking advantage of Microsoft SQL server environments.
- MicroBurst: A set of PowerShell tools for attacking and checking Microsoft Azure environments.
- PowerHuntShares: A program that helps you find and look at SMB file shares in Active Directory environments that are too open.
- Inveigh Zero: A tool for Windows that lets you spoof and act as a man-in-the-middle.
- RayV Lite: A tool for hacking hardware to take advantage of microships.
